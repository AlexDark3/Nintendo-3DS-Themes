# 🎨 Nintendo 3DS Themes Collection

Repositorio dedicado a la recopilación y creación de **temas personalizados para Nintendo 3DS**.
Aquí encontrarás diseños únicos para darle un estilo diferente a tu consola.

---

## 📦 Contenido

* Temas `.zip` listos para usar
* Archivos de personalización (BGM, iconos, fondos)
* Versiones organizadas por categorías

---

## 📁 Estructura del repositorio

```
themes/
│
├── anime/
├── videojuegos/
├── minimalistas/
├── personalizados/
└── otros/
```

---

## ⚙️ Requisitos

Para usar estos temas necesitas:

* Una Nintendo 3DS con **CFW (Custom Firmware)** instalado
* Aplicación recomendada:

  * Anemone3DS

---

## 📥 Instalación

1. Descarga el tema que quieras desde este repositorio
2. Extrae el archivo `.zip` (si es necesario)
3. Copia la carpeta del tema en:

```
/themes/
```

(en la raíz de tu tarjeta SD)

4. Inserta la SD en tu consola
5. Abre **Anemone3DS**
6. Selecciona el tema
7. Presiona **A → Install Theme**

---
